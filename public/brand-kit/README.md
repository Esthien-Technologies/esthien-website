# Esthien Labs Brand Kit

Selected direction: Silicon Strata, refreshed with a more linear E mark.

## Files

- `esthien-logo-horizontal.svg` - primary dark-background lockup.
- `esthien-logo-horizontal-light.svg` - primary light-background lockup.
- `esthien-logo-horizontal-monochrome.svg` - single-color lockup for print, engraving, and restricted production.
- `esthien-mark.svg` - compact dark-background mark.
- `esthien-mark-light.svg` - compact light-background mark.
- `esthien-mark-monochrome.svg` - single-color compact mark.
- `esthien-social-avatar.svg` - square profile image for official social channels.
- `brand-tokens.css` - reusable color and typography variables.
- The website-level `esthien-og.svg` and `esthien-og.png` files provide the editable and deployment-ready social sharing artwork.

## Tagline

Custom FPGA chipsets for physical intelligence.

Use the tagline in title case in prose and as `CUSTOM FPGA CHIPSETS FOR PHYSICAL INTELLIGENCE` in logo lockups or compact identity applications.

## Color Tokens

- Core black: `#0A0D10`
- Panel black: `#11181C`
- Lab white: `#F8FAF9`
- Logic green: `#32D69A`
- Signal cyan: `#4CC9F0`
- Graphite line: `#2B373D`

## Type Direction

- Wordmark: heavy geometric sans, high confidence, wide technical presence.
- Supporting text: clean sans with tracked uppercase for semiconductor/instrument feel.
- Editorial accent: restrained serif only for larger brand statements.
- Monogram: the E is built from one vertical stem and three aligned horizontal strokes. Keep this geometry consistent across SVG, PNG, social, and website UI formats.

## Usage Notes

- Prefer the dark lockup on black or graphite and the light lockup on white or neutral gray.
- Do not stretch, rotate, outline, recolor, or add shadows to the logo.
- Keep clear space around the logo at least equal to the height of the green strata bar.
- Use the standalone mark only when the full `ESTHIEN LABS` wordmark would be too small.
- Keep the company name written as `Esthien Labs` in prose and `ESTHIEN LABS` inside the identity system.
- Do not recreate the wordmark with a substitute font; use the supplied SVG artwork.
- Use the source SVGs when exporting new PNGs so the linear E, signal bars, and tagline remain consistent.

## Minimum Size

- Horizontal lockup: 180 px wide on screens or 38 mm in print.
- Standalone mark: 24 px on screens or 6 mm in print.

## File Naming

Use lowercase kebab-case for exported assets: `esthien-[asset]-[variant].[ext]`.
